===============================
Purchase order project template
===============================

* If when confirming a purchase order it has a project template defined, a
  project will be created automatically for said purchase order.
* In Purchase order form new action "Create project" for purchase orders that
  are not in draft status.

Bug Tracker
===========

Bugs are tracked on `GitHub Issues
<https://github.com/avanzosc/odoo-addons/issues>`_. In case of trouble, please
check there if your issue has already been reported. If you spotted it first,
help us smash it by providing detailed and welcomed feedback.

Do not contact contributors directly about support or help with technical issues.
